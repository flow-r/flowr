Building pipelines¶

A working example¶

## 6 jobs with 10 each
cmds = sprintf("sleep %s", round(rchisq(6*10, df = 1), 1))
nms = sprintf("myjob%s", rep(1:6, each = 10))
mat = data.frame(cbind(jobname = nms, cmd = cmds), stringsAsFactors = FALSE)
head(mat)

#>   jobname       cmd
#> 1  myjob1 sleep 1.3
#> 2  myjob1   sleep 0
#> 3  myjob1 sleep 0.6
#> 4  myjob1 sleep 1.3
#> 5  myjob1 sleep 1.3
#> 6  myjob1   sleep 0
